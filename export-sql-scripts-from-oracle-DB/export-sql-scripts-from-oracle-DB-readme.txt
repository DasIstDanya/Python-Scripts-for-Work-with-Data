Скрипт выгружает все доступные пользователю базы данных (БД) oracle объекты sql следующих видов: типы, триггеры, функции, процедуры, триггеры.
Я использую данный скрипт для быстрого бэкапа данных из схемы. Для этого файл помещен непосредственно в репозитории гитхаба, где должны находиться папки с этими объектами БД.
Обычно скрипт я запускаю из pycharm, там же настроена синхронизация с gitlab по ssh.
Для успешной работы скрипта необходимо знать и ввести параметры подключения к базе, такие как хост, порт, сервисное имя, логин и пароль, имя владельца
